# PathToAIJob

# 前言

因为立志于为人工智能（Artificial Intelligence，简称AI）领域的发展做贡献，于是希望能用写技术博客和大家分享的方式监督自己，保持学习新知识的良好习惯。同时，也顺便早早为后面的找工作做准备。

希望深爱AI的小伙伴们都能共同进步，找到理想的工作，为推动AI理论的发展做出自己的一份努力，让技术造福于人类，让计算机帮我们一起探索世界的终极奥秘！
